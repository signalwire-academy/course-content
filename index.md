# SignalWire AI Agents Certification

Welcome to the official certification program for building voice AI agents with the SignalWire AI Agents SDK.

## Program Overview

This certification program teaches you to build production-ready voice AI agents. The program is divided into three levels:

| Level | Focus | Duration |
|-------|-------|----------|
| **Level 1** | Foundations | 2-3 weeks |
| **Level 2** | Intermediate Development | 3-4 weeks |
| **Level 3** | Advanced & Enterprise | 4-6 weeks |

## Getting Started

1. [Set up your development environment](getting-started)
2. Start with [Level 1: Foundations](level-1/)
3. Complete labs and assessments to progress

## What You'll Build

By completing this program, you'll be able to:

- Create voice AI agents that handle phone calls
- Build natural conversational experiences
- Integrate with external APIs and databases
- Deploy production-ready voice applications
- Implement security and compliance requirements

## Requirements

- Python 3.10 or higher
- Basic programming knowledge
- A SignalWire account ([sign up free](https://signalwire.com))
