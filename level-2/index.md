# Level 2: Intermediate

Build on your foundation with intermediate concepts and real-world integrations.

## What You'll Learn

- Advanced function results and actions
- DataMap integrations for external data
- The Skills system for reusable components
- Conversation contexts and workflows
- State management patterns
- Call recording and transcription
- Call transfers and multi-agent systems
- Production deployment strategies

## Prerequisites

- Completed Level 1 certification
- Familiarity with REST APIs
- Basic understanding of async programming
